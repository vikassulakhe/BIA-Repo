# Travel-Planner

Streamlit app that uses Groq to search destinations, generate itineraries, and provide mock flight/hotel options. Bookings are simulated and stored in session state.

Setup

1. Copy `.env.example` from root or set `GROQ_API_KEY` in your environment:

```bash
export GROQ_API_KEY="your_real_key"
# or copy .env.example -> .env and edit
```

2. Run the app:

```bash
cd BIA-Repo/Travel-Planner
streamlit run app.py
```
