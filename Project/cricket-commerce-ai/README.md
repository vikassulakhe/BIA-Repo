# 🏏 Cricket Commerce AI Agent

Enterprise-style AI support chatbot for a cricket e-commerce store.

Uses:
- Ollama
- Qwen3.5 9B
- Streamlit UI

## Features

- Product inquiry agent
- Order tracking agent
- Refund workflow agent
- LLM-based intent classification
- Local LLM (no API costs)
